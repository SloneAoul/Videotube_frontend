# VideoTube - Frontend

YouTube-like video platform built with React.js and Tailwind CSS.

## Live Demo
https://videotube-frontend-beta.vercel.app/

## Features
- Video upload and streaming
- User authentication and protected routes
- Comments, likes, and channel management
- Responsive UI across all devices

## Tech Stack
- React.js
- Tailwind CSS
- Axios
- React Router

## Setup
npm install
npm run dev
